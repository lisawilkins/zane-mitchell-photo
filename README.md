# Zane Mitchell — Rodeo & Western Photographer

Portfolio site for Zane Mitchell, a documentary photographer capturing the raw, unfiltered spirit of the American West — from the arena to the open range.

## Live Site

https://lisawilkins.github.io/zane-mitchell-photo/

## Categories

- **Rodeo** — Mounted shooting, barrel racing, bull riding, bronc riding, team roping, and more
- **Race Track** — On track, warmup, backstage, winners circle, riders

## Tech

Static HTML/CSS/JS — no build tools or dependencies required.

## Local Preview

```bash
npx serve .
```

Then open http://localhost:3000

## Adding Photos

1. Drop images into `assets/img/`
2. Add a `.photo-card` entry in `index.html` with the appropriate `src` and `data-cat` attributes
3. The placeholder gradient hides automatically once a real image loads
